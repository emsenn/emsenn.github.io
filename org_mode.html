Org-mode is a text-formatting language and its interpretter software, a part of the Emacs text editor.

Whenever I'm using a computer, I'm probably using Org-mode. I use it to keep notes, draft essays, keep ledgers, write software, and more.

Components

Org-agenda

Org-mode agenda is a part of Org-mode used for handling a temporal schedule.

Org-links

An Org-mode link is a hypertext reference to another location within a computer file.

Org-attach-screenshot

Org-attach-screenshot is an Emacs package, part of the Org ecosystem, that is for using Org-attach to link screenshots to Org-mode entries.

Babel

Babel is a part of the Org ecosystem that allows for executing source code within Org-mode files.

Org-capture

An Org-mode capture template is a template for text input automation within Emacs.
