Org-mode structure editing

Key sequence Command Description
M-RET (org-meta-return) Insert a new heading, item, or row.
C-RET (org-insert-heading-respect-content) Insert a new heading at the end of this entry.
C-u C-u M-RET (org-meta-return) Insert a new heading at the end of this entry's parent.
M-S-RET (org-insert-todo-heading) Insert a new task entry at point.
C-S-RET (org-insert-todo-heading-respect-content) Insert a new task at thhe end of this entry.
M-LEFT (org-do-promote) Raise a headline to the parent level.
M-RIGHT (org-do-demote) Lower a headline to the child level.
M-UP (org-move-subtree-up) Move entry up within its current level.
M-DOWN (org-move-subtree-down) Move entry down within its current level.
C-c C-x C-w (org-cut-subtree) Remove an entry from the buffer and place it in the kill "ring"
C-c C-x C-w (org-copy-subtree Place a copy of the entry in the kill "ring"
C-c C-x C-y (org-paste-subtree) Places an entry from the kill "ring" into the current parent entry
C-c C-w (org-refile) Provides an interface for moving an entry to a new location
C-c ^ (org-sort) Sort all entries within a parent.
Key sequence Command Description
S-UP (org-shiftup) Move to the previous item
S-DOWN (org-shiftdown) Move to the next item
M-UP (org-metaup) Move the current item up
M-DOWN (org-metadown) Move the current item down
M-LEFT (org-metaleft) Increase the identation of the current item
M-RIGHT (org-metaright) Decrease the indentation of the current item
M-S-LEFT (org-shiftmetaleft) Increase the indentation of the current item and its children
M-S-RIGHT (org-shiftmetaright) Decrease the indentation of the current item and its children
C-c C-c (org-ctrl-c-ctrl-c) Reflow list and complete current item, if it has a checkbox
Key sequence Command Description
C-UP (org-backward-paragraph) Go to the previous paragraph
C-c C-n (org-next-visible-heading) Go to next heading
C-c C-p (org-previous-visible-heading) Go to previous heading
C-c C-f (org-forward-heading-same-level) Go forward a heading at current depth
C-c C-b (org-backward-heading-same-level) Go backward a heading at current depth
C-c C-u (outline-up-heading) Go up to the parent depth's headline
