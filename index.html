emsenn's digital garden

My name is emsenn and this is my digital garden. I first started cultivating it on December 16th, 2020. It is a virtual space where each place is a hypertext document. Documents point to each other and other resources. Attached to each page is a list of "backlinks:" other documents in this and other spaces that point to that document. You can see that this page backlinks to the Records Index, because that document mentions this one. (That document, in fact, mentions every document in the garden, and can serve as a generalized navigation space.)

If you're viewing this "garden" on the Web, please note that some records might not exist, and records that do exist might have sections redacted. This garden is cultivated in private using Groundhog Emacs, and only select records are exported to the public Website.
