Introduction

Ox-org

Ox-org is a built-in part of the Org ecosystem that allows for the export of Org-mode format files into Org-mode, itself.

Use

To have your Emacsen enable Org-mode export, use the following declaration: 

(use-package ox-org)
Table 1: A table of key sequences used to operate Ox-org.
Key sequence Command Description
C-c C-e S-o o (org-export-dispatch) Export the current buffer as an Org-mode file.
C-c C-e S-o S-o (org-export-dispatch) Export the current buffer as an Org-mode buffer.
